Tecnologiga

Tecnologia é um termo amplo que se refere a um conjunto de conhecimentos, ferramentas, técnicas e dispositivos utilizados para criar, manipular e resolver problemas em diversos campos.

Esse área como muitos dizem é a área do futuro, já que nos ultimos tempos tudo está voltado para internet.daqui a alguns anos tudo vai ser baseado na Inteligência Artificial.

Áreas que a tecnologia está presente:

  1. Programação
  2. Cloud Computing
  3. Programação mobile
  4. Web Design
  5. Administração de redes
  6. Segurança da informação

Profissões digitais

  • Desenvolvedor de Software
  • Engenheiro de dados
  • Especialista em Inteligência Artificial
  • Cientista de Dados
